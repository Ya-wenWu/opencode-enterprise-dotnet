# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you discover a security vulnerability, please report it through GitHub's **Private Vulnerability Reporting** rather than opening a public issue.

To report a vulnerability, go to:

[https://github.com/Ya-wenWu/opencode-enterprise-dotnet/security/advisories](https://github.com/Ya-wenWu/opencode-enterprise-dotnet/security/advisories)

You can expect to receive a response within **48 hours**. We will keep you informed of the progress toward resolution and may reach out for additional information if needed.

Thank you for helping keep our project and its users safe.
